## H. Dubner ([Wikipedia](https://en.wikipedia.org/wiki/Harvey_Dubner))

## Clifford A. Pickover ([Wikipedia](https://en.wikipedia.org/wiki/Clifford_A._Pickover))

**Clifford**:
- *cliff* = Klippe
- *ford* = Furt (Flussdurchgang)
- → „Furt an der Klippe"

**Pickover**:
- *pick* = Spitze / hacken / auswählen
- *over* = über
- Schwer eindeutig, evtl.:
  - „der hinauswählt"
  - Oder ursprünglich Orts-/Berufsname

**Interpretation**:
- Kombination aus Landschaft + Handlung
- Klingt fast wie „jemand, der Übergänge/Schwellen überwindet"
